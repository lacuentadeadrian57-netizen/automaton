# Conway's Game of Life - Automaton

This project is an implementation of Conway's Game of Life in **HTML, CSS, and JavaScript**.  
It simulates cellular automata on a grid, where each cell evolves according to simple rules based on its neighbors.

---

## 🚀 Features
- Interactive grid where you can toggle cells on/off by clicking.
- Play/Pause button to control the simulation.
- Forward step button to advance one generation at a time.
- Adjustable grid size (rows and columns).
- Visual representation using CSS grid.
